utm_to_wgs84¶

pycraf.geospatial.utm_to_wgs84(ulon, ulat, zone, south=False)[source]¶

Convert UTM coordinates to GPS/WGS84.

Parameters:

ulon, ulat : Quantity

UTM longitude and latitude [m]

zone : int

UTM zone (e.g., 32 for longitudes 6 E to 12 E); see Wikipedia for more information on the zones

south : bool

Hemisphere; set to True for southern hemisphere (default: False)
Returns:

glon, glat : Quantity

GPS/WGS84 longitude and latitude [deg]

Notes

  • Uses

    +proj=utm +zone=xx +ellps=WGS84 +datum=WGS84 +units=m +no_defs [+south]

    for pyproj setup (inverse transformation). Only one zone per function call is allowed.

  • This function uses only the longitudal zone scheme. There is also the NATO system, which introduces latitude bands.
