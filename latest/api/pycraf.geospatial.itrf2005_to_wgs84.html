itrf2005_to_wgs84¶

pycraf.geospatial.itrf2005_to_wgs84(x, y, z)[source] [edit on github]¶

Convert ITRF2005 coordinates to GPS/WGS84.

ITRF is the International Terrestrial Reference System. (A geocentric coordinate system.)

Parameters:

x, y, z : Quantity

ITRF2005 geocentric coordinates, (x, y, z) [m]
Returns:

glon, glat : Quantity

GPS/WGS84 longitude and latitude [deg]

height : Quantity

Height (amsl) [m]
