cispr11_limits¶

pycraf.protection.cispr11_limits(freq, detector_type='RMS', detector_dist=<Quantity 30. m>)[source] [edit on github]¶

Industrial-device limits for given frequency and distance from device according to CISPR-11 (EN 55011).

Parameters:

freq : Quantity

Frequency or frequency vector [Hz]

detector_type : str, optional

Detector type: ‘QP’ (quasi-peak), ‘RMS’ (default: ‘RMS)

detector_dist : Quantity, optional

Distance between source and detector [m], (default: 30 m)
Returns:

efield_limit : Quantity

Electric-field limit permitted by CISPR-11 at given distance. [dB(uV^2 / m^2)]

meas_bw : Quantity

Measure bandwidth (of detector) [kHz]
